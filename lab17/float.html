Challenge

The challenge for this lab is to create a grid layout, css layout, flexboard layout, and float layout. We are to create four different htmls for the four different layouts, as well as a different css for each one.

Problems

I didn't have too many issues with this layout, it was pretty simple since I'm already a bit familiar with float elements. I just had an issue with formatting the footer, and there's still a tiny block that is on the bottom on my website that I have no idea how to get rid of.

Final Result

The results are simple, nothing too surprising since it's just float elements. I like how much more spaced out this layout is rather than having the boxes so close to each other, but again, this is just a simple basic layout.
